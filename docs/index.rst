Global Forecast Validation Tools
================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   python_functions
   cli

The global-forecast-validation package is a tool used to validate forecasts that have been generated with the
`RAPIDpy <https://github.com/BYU-Hydroinformatics/RAPIDpy>`_ package. It contains a few python functions that are
simple to use but powerful in generating forecast metrics and skill scores in large regions of streams.
